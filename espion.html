ESPION



Espion is an futuristic IOT edge device which can be customized to work with variety of Nodes. Our uniqueness is that we use master slave approach, the hub will be the master and the sensor nodes will be act as slaves which will reduce the requirement of data usage in terms of mobile communication and storage. This requires only the central hub to be connected to the internet, data transfer between nodes and hub will take place over WLAN. The central hub will act as storage of data in case of connectivity failure. Espion is an complete IOT hardware which can be expandable and customized for any needs and it can be connected to any IOT software through their APIâ€™s so that all the data passes through the Espion can be pushed to the target Software platform.
